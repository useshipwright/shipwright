## Quick start

```bash
cd service
docker compose up -d --build
bash scripts/smoke-test.sh
```

12 checks run against 4 endpoints using real tokens from the Firebase Auth Emulator. No credentials needed.

## What this is

A complete build: PRD went in, a production service came out, with tests, ADRs, and a deploy script. The service verifies Firebase ID tokens, batch-verifies up to 25 tokens (5 concurrent SDK calls), and looks up user profiles. Fastify + TypeScript, deployed to Cloud Run. The dashboard was built by hand to showcase the service.

## Deploy to GCP

```bash
bash scripts/deploy.sh
```

Creates the GCP project, enables APIs, builds both containers, deploys the service and dashboard to Cloud Run. Interactive: prompts for project ID and billing account.

After deploy, enable Email/Password and Google sign-in in the Firebase Console, and add the dashboard URL to authorized domains.
