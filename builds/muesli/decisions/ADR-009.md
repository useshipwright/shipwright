# ADR-009: PRD Correction: CAP-7: Search

## Context

PRD validation found a mismatch between the PRD and domain documentation.

**PRD claims:** Embedded via Vertex AI (text-embedding-004) or configurable local endpoint (EMBEDDING_ENDPOINT)

**Documentation says:** Vertex AI docs note text-embedding-004 is being deprecated (referenced in Genkit deprecation issue). The recommended models are now gemini-embedding-001 (newest), text-embedding-005, or text-multilingual-embedding-002. Using text-embedding-004 for a new project is inadvisable.

**Source:** gcp-vertex-ai — overview.md

## Decision

Follow the domain documentation behavior, not the PRD claim. The implementation should: Vertex AI docs note text-embedding-004 is being deprecated (referenced in Genkit deprecation issue). The recommended models are now gemini-embedding-001 (newest), text-embedding-005, or text-multilingual-embedding-002. Using text-embedding-004 for a new project is inadvisable.

## Consequences

- PRD section 'CAP-7: Search' is overridden
- Implementation follows verified API behavior
- Reduces risk of runtime errors from incorrect assumptions
